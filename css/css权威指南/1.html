块级元素

一、width,height

this is a p

当内容大于宽度时，则换行

当内容不满足高度时，则用空白补充

二、margin、padding、border

计算下面div显示的height，为第一个元素的上边框到最后一个元素的下边框的距离，如果上下两个元素都使用了margin，那么它们之间的距离为两个margin中较大的那个

div的width，为子元素的水平方向“七属性”的和，即margin-left,border-left,padding-left,width,padding-right,width,padding-right,border-right,margin-right。如果出现了子元素大于父元素，可能会出现元素突出的效果
