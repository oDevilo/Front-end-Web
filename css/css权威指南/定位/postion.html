z-index

默认是0，z-index:-1 表示更低优先级，1表示更高，更高显示于更上层

absolute

生成绝对定位的元素，相对于 static 定位以外的第一个父元素进行定位。 元素的位置通过 "left", "top", "right" 以及 "bottom" 属性进行规定。

这是带有绝对定位的标题

通过绝对定位，元素可以放置到页面上的任何位置。 下面的标题距离页面左侧 100px，距离 static 定位以外的第一个父元素水平方向 150px。 这里，上一个div没有使用定位，所以默认是static，则不满足会再找上级，最终找到body，以body为主

relative

生成相对定位的元素，相对于其正常位置进行定位。因此，"left:20" 会向元素的 LEFT 位置添加 20 像素。

这是位于正常位置的标题

这个标题相对于其正常位置向左移动样式 "left:-20px" 从元素的原始左侧位置减去 20 像素。

这个标题相对于其正常位置向右移动样式 "left:20px" 向元素的原始左侧位置增加 20 像素。

fixed

生成绝对定位的元素，相对于浏览器窗口进行定位。 元素的位置通过 "left", "top", "right" 以及 "bottom" 属性进行规定。

inherit

规定应该从父元素继承 position 属性的值。
